How to Set Up Firebase
  1. Go to Firebase Console
  2. Create a new project or use an existing one
  3. In Project Settings, add a new Web App
  4. Copy the Firebase configuration values into the fields above
  5. In Firebase Console, go to Storage and set up Firebase Storage
  6. Set Storage rules to allow read and write access
